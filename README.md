RequireJS-Dojo-AngularJS
========================

Just use requireJS to load dojo and angularjs example

#### 目录结构
```shell
├── bower_components
│   ├── angular
│   ├── bower.md
│   ├── dijit
│   ├── dojo
│   ├── domReady
│   └── requirejs
├── bower.json
├── index.html
├── js
│   ├── app.js
│   ├── bootstrap.js
│   ├── controllers
│   ├── directives
│   └── main.js
├── node_modules
│   ├── bower
│   └── http-server
├── package.json
└── README.md
```
#### 如何使用

- git clone https://github.com/hjzheng/RequireJS-Dojo-AngularJS 
- npm start
